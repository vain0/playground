Launch VSCode on Docker to develop F#

## Usage

On Ubuntu:

- Install Docker
- Download (or git-clone) this repository

```sh
cd fsharp
docker-compose up
```

VSCode instance will appear.

Press F5 to launch a debugger to print "Hello World" in debug console.
