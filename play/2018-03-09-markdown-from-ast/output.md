# Hello world!

Sample codes.

- There are *two* version of typical **C++** hello worlds.
- iostream ver:
    ```cpp
    // C++ Hell World (using iostream)
    #include <iostream>

    void main() {
        std::cout << "Hello, world!" << std::endl;
    }
    ```
-
    - printf ver:
    -
        ```cpp
        // C++ Hello World (using printf)
        #include <cstdio>

        void main() {
            std::printf("Hello, world!\n");
        }
        ```
